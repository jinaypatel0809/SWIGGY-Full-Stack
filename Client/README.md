# 🍔 Swiggy Clone — Home Page

Full Swiggy-inspired Home Page built with **React + Tailwind CSS**.

## 📦 Components

| Component | File | Section |
|-----------|------|---------|
| Navbar | `src/components/Navbar.jsx` | Top navigation |
| Hero | `src/components/Hero.jsx` | Orange hero banner |
| FoodCategories | `src/components/FoodCategories.jsx` | Category grid |
| RestaurantCards | `src/components/RestaurantCards.jsx` | Dineout cards |
| CitiesSection | `src/components/CitiesSection.jsx` | City links |
| Footer | `src/components/Footer.jsx` | Footer |
| Home | `src/pages/Home.jsx` | Assembles all sections |

## 🚀 Run

```bash
npm install
npm run dev
```

Open: http://localhost:5173
